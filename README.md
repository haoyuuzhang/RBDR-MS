# 工业运营推演与智能决策平台

---

## 项目定位
面向制造企业、产业园区与政府部门，建设一个轻量、可扩展、可接入AI的工业运营推演与智能决策底座。
平台不定位于传统重型3D仿真软件，而是围绕生产线、订单、设备、库存、物流与突发扰动，提供 **“快速建模—仿真推演—优化重规划—AI辅助解释”** 的闭环能力。

---

## 项目背景
- **企业**——需要低门槛的 what-if 推演：设备停机、订单插单、物料延期、产能瓶颈等问题能快速形成多方案比较。
- **政府与园区**——需要产业韧性评估工具：用于智能制造示范、园区运营分析、重点产业链风险推演。
- **科研**——需要统一实验底座：支撑运筹优化、多智能体协同、AI辅助决策与扰动恢复算法验证。

---

## 两条建设主线
| 工程平台搭建 | 科研算法探索 |
| :--- | :--- |
| 轻量离散事件仿真、生产线/仓储/物流模板、扰动注入、运行日志与可视化报告。 | 围绕排产优化、资源重配置、扰动恢复、多智能体协同与触发式重规划开展算法研究。 |
| 预留企业数据接入、优化器接口、AI模块接口，形成可演示、可复用的原型平台。 | 结合AI/LLM做场景配置辅助、方案解释、异常诊断、规则生成与决策报告生成。 |

---

## 近期任务需求
1.  完成一个可运行 MVP：参数化生产线建模、订单流转、设备故障/物料延期注入、关键指标统计。
2.  实现至少一种优化重规划能力：如插单重排、故障恢复、瓶颈诊断或多方案成本—交期比较。
3.  形成可用于宣讲的演示案例：输入一个扰动，输出受影响订单、瓶颈节点、恢复方案与解释报告。
